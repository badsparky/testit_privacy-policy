  <h1>プライバシーポリシー</h1>
  <p>当アプリは、ユーザーのプライバシー保護を重視しています。以下は、当アプリのプライバシーポリシーになります。</p>

  <h2>1. 収集される情報</h2>
  <p>当アプリは、ユーザーがアプリを使用する際に提供する情報を収集しません。</p>

  <h2>2. ストレージとメディアの使用</h2>
  <p>当アプリは、アプリの実行に必要な情報のみを保持します。この情報は、ユーザーの許可なく第三者に一切共有されません。</p>

  <h2>3. ネットワークの使用</h2>
  <p>当アプリは、アプリの実行に必要な場合にのみ、インターネットに接続してデータを取得します。取得したデータは、アプリ内のみに保存され、第三者に一切共有されません。</p>

  <h2>4. 広告の表示</h2>
  <p>当アプリは、広告の表示にGoogle AdMobを利用しています。広告表示に必要な情報を取得するために、AdMobが提供する広告IDを取得します。取得した広告IDは、広告表示のためにのみ使用され、第三者に一切共有されません。</p>

  <h2>5. 情報の保護</h2>
  <p>当アプリは、適切な安全対策を実施し、ユーザーの情報を保護します。また、万一、情報の漏洩や不正アクセスが発生した場合は、速やかに対処し、その原因の究明と再発防止に努めます。</p>

  <h2>6. 法令、規範の遵守</h2>
  <p>当アプリは、適用される法令、規範を遵守します。また、プライバシーポリシーに関する変更があった場合には、適切な方法でユーザーに通知するものとします。</p>
